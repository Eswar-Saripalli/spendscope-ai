# SpendScope AI

Live Demo: [https://spendscope-ai-chi.vercel.app/](https://spendscope-ai-chi.vercel.app/)

## Overview

SpendScope AI is a modern SaaS-style dashboard that helps startups and small teams analyze their AI tool spending, identify unnecessary costs, and discover optimization opportunities.

The platform provides:

* AI spend analysis
* Savings recommendations
* Analytics visualizations
* PDF export reports
* Persistent audit history
* Responsive dashboard UI

---

# Features

## AI Spend Audit

Analyze spending across multiple AI tools including:

* ChatGPT
* Claude
* Gemini
* Cursor
* GitHub Copilot

---

## Smart Recommendations

The dashboard detects unnecessary or inefficient spending and suggests optimization opportunities.

Examples:

* Downgrade recommendations
* Smaller team plan suggestions
* Cost reduction opportunities

---

## Analytics Dashboard

Visualize spending using interactive charts.

Includes:

* Current spend
* Optimized spend
* Yearly savings
* Spend comparison chart

---

## PDF Export

Generate downloadable audit reports containing:

* Current spend
* Optimized spend
* Savings summary
* Recommendations

---

## Audit History

All audits are stored locally in the browser using localStorage.

Users can:

* View previous audits
* Track savings history
* Clear audit history

# Installation

## Clone Repository

```bash
git clone https://github.com/Eswar-Saripalli/spendscope-ai.git
```

## Navigate to Project

```bash
cd spendscope-ai
```

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```
